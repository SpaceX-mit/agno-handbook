MiniMax 使用 - 三步走
====================

步骤 1: 获取密钥
---------------
访问: https://platform.minimax.io/
注册 → 创建 API 密钥 → 复制保存

步骤 2: 设置密钥
---------------
export MINIMAX_API_KEY='sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'

步骤 3: 启动 Agent
-----------------
cd /home/bianbu/agno-riscv64
./cookbook/system_agent/start_minimax.sh

就这么简单！
